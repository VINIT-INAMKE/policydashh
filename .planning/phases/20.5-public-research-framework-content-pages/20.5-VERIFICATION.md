### SC-1 Section Naming — Clarification Note

ROADMAP SC-1 states: "renders executive summary, current Indian landscape, key gap clusters". The implementation uses section headings "Overview", "Key Themes", "Research Outputs" — names specified in `20.5-UI-SPEC.md` §"Research Page Section Structure" (lines 315-322), which is the authoritative planning document. Content mapping:

- "executive summary" → hero h1 "Understanding the Landscape" + lead paragraph
- "current Indian landscape" → `#overview` section body (regulatory environment, global benchmarks, tech capabilities)
- "key gap clusters" → `#key-themes` section (legal classification, taxation, data governance, use cases, institutional barriers)
- "downloadable research report" → `#outputs` section with PDF download anchor

This is label drift between ROADMAP description and UI-SPEC implementation — the content substance is present. The UI-SPEC was produced by `/gsd:ui-phase` after the ROADMAP and takes implementation precedence per D-22 in CONTEXT.md. No gap.

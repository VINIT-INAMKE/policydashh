## Decisions Made

- **font-semibold over font-medium in ResearchTocAside:** UI-SPEC declares a 2-weight typography contract (normal 400 + semibold 600). Existing `PublicSectionNav` uses `font-medium` but is explicitly out of scope for this phase; the new component complies.
- **Binary PDF via node -e Buffer/latin1:** no new dependency needed. Git's `core.autocrlf` emits an LF→CRLF warning at add-time, but the blob stored in the object database is byte-identical to the on-disk file (verified via `git show HEAD:... | wc -c` = 584). The `astextplain` diff attribute only affects human-readable diffs, not storage.
- **Server Component for the page, client island for TOC:** RSC composition pattern — the page streams statically from server, only the 4-item TOC is hydrated for IntersectionObserver. Zero auth hydration, matches Phase 9 public portal baseline.
- **Ran parallel with Plan 20.5-01 using --no-verify commits:** explicit instruction from parallel_execution context to avoid pre-commit hook contention; orchestrator will run hooks once after the wave completes.

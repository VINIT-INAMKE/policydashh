       - `resolve`: Set `resolved = true` on commentThread. Permission: feedback:create (any authenticated).
       - `reopen`: Set `resolved = false` on commentThread. Permission: feedback:create.
       - `delete`: Delete commentThread by id. Permission: only author (compare ctx.userId === thread.authorId) or admin.
